# 🚀 CHAFO-TIME-MANAGEMENT (Infinite Edition)

**CHAFO OS v4** is a futuristic, high-performance productivity dashboard designed for users who demand deep focus and real-time efficiency tracking. This system is tailored for engineering students managing rigorous academic workloads.

## ✨ Key Features
*   **Dynamic Task Management**: Easily create, prioritize, and categorize daily objectives.
*   **Focus Mode**: Integrated Pomodoro timer to facilitate deep concentration during study sessions.
*   **Efficiency Analytics**: Real-time tracking of productivity rates, completion statistics, and daily streaks.
*   **Glassmorphic UI**: A modern dark-mode interface featuring smooth CSS animations and a sleek aesthetic.
*   **Data Sovereignty**: Built-in functionality for data backup (Export) and restoration (Import) using JSON files.

## 🛠️ Tech Stack
*   **Frontend**: HTML5 and CSS3 (utilizing Custom Variables & Glassmorphism effects).
*   **Logic**: Vanilla JavaScript (ES6+) for high-speed client-side processing.
*   **Storage**: LocalStorage API to ensure data persistence across browser sessions.

## 📖 How to Use
1.  Open `index.html` in any modern web browser.
2.  Input your task in the "Define your next objective" field.
3.  Select a Category (e.g., School Work, Personal) and set the Priority level.
4.  Click **INITIALIZE** to start the task.
5.  Activate **Focus Mode** (25-minute timer) for optimized time management.
